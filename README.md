# MoodBoard Platform 

**Moodboard Platform** — это современная веб-платформа для визуализации и обмена вдохновляющими коллекциями. Проект представляет собой сайт-каталог, где администраторы публикуют тематические мудборды, а обычные пользователи могут просматривать и искать их по тегам и ключевым словам.

## О проекте

### Концепция
Платформа сочетает в себе эстетику визуального контента с удобством навигации. Администраторы создают и управляют мудбордами, добавляя изображения, описания и теги, в то время как пользователи могут легко находить интересующие их коллекции через интуитивно понятную систему поиска.

### Ключевые возможности
- **Публичный каталог** — просмотр всех опубликованных мудбордов
- **Поиск** — быстрый поиск по названию и тегам
- **Админ-панель** — управление мудбордами с авторизацией
- **Теговая система** — категоризация контента для удобной навигации
- **Загрузка изображений** — поддержка загрузки и хранения медиафайлов

## Технологический стек

### Бэкенд
- **FastAPI** — современный фреймворк для создания высокопроизводительных API
- **PostgreSQL** — надежная реляционная база данных
- **SQLAlchemy** — ORM для работы с базой данных
- **JWT (JSON Web Tokens)** — безопасная аутентификация администраторов
- **Argon2** — современный алгоритм хэширования паролей
- **Pydantic** — валидация данных и автоматическая документация
- **Uvicorn** — ASGI сервер для запуска приложения

### Фронтенд
- **React 18** — библиотека для создания пользовательских интерфейсов
- **Vite** — быстрый инструмент сборки и разработки
- **React Router DOM** — маршрутизация в приложении
- **Axios** — HTTP-клиент для взаимодействия с бэкендом
- **SCSS** — препроцессор для стилизации

### База данных
- **PostgreSQL** — основная база данных
- **SQLAlchemy** — ORM для управления данными

### Менеджеры зависимостей
- **pip** — для управления зависимостями Python
- **npm** — для управления зависимостями JavaScript
